Название проекта: бот Тамерлана
Суть проекта: бот, предоставляющий доступ к chat-gpt и к нескольким полезным функциям.
Использованные технологии:
python
sql
g4f(Бесплатное api для доступа к gpt)
python-telegram-bot
Количество строк кода: 300
Количество файлов: 5
Модули:
main - запускает бота
gpt - отвечает за взаимодействие с gpt
db - работа с бд
commands - все команды бота
config - хранит в себе токен бота и прокси
